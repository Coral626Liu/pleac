PLEAC - Programming Language Examples Alike Cookbook

Overall

Being a big fan of the Perl Cookbook (by Tom Christiansen & Nathan Torkington, published by O'Reilly), this project aims to gather fans of programming, in order to implement the solutions in other programming languages.

I hope this project can become a primary resource for quick, handy and free reference to solve most common programming problems using higher-level programming languages, and for comparison on ease-of-use and power/efficiency of these languages.

To contribute to this projet, you only need to implement the Solutions shown in the Perl Cookbook in your language of choice, writing them in a simple description language (look at the "code" links in the "Status" section for examples). Actually I have an sgml skeleton, and a small Ruby program to generate a sgml source from the skeleton and the Solutions; this sgml source then gets compiled into html and ps output. The material, considered as some Documentation, is wholy released under the Gnu Free Documentation License.

Please subscribe to the discussion mailing-list if interested in the project.

Building the project

To generate the sgml and build the html and ps outputs off the sources, you'll need the relevant building software. For example, with Linux-Mandrake 8.0, here's the list of the packages needed: 

make-3.79.1-5mdk.i586.rpm
ruby-1.6.2.2-2mdk.i586.rpm
sgml-common-0.2-4mdk 
sgml-tools-1.0.9-19mdk 
openjade-1.3-10mdk 
docbook-style-dsssl-1.64-1mdk 
docbook-dtd31-sgml-1.0-3mdk 
jadetex-3.6-1mdk 
tetex-1.0.7-21mdk 
tetex-dvips-1.0.7-21mdk 
tetex-latex-1.0.7-21mdk

Once the CVS snapshot is untarballed, just issue make (or just make html for example).
